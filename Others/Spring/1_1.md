## spring是啥

- 一个java EE开源框架
- 以IOC和AOP为内核

## spring的体系就结构

spring有20个模块，分为：

- Core Container(核心容器)		--Beans,Core,SpEL,Context-support,Context
- Data Access/Integration(数据库访问/集成)		--JDBC,ORM,OXM等	
- Web		--Servlet，Socket等
- AOP
- Instrumentation
- Messaging
- Test
这几个部分
